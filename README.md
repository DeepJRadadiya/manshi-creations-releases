# Manshi Creations - App Releases

This repository contains the official releases of the **Manshi Creations** mobile app.


---

## 📥 Download Latest Version

Go to the [Releases](../../releases) section and download the latest `app-release.apk` file.

---

## 📲 How to Install

1. Download the latest `app-release.apk` from the [Releases](../../releases) page
2. Open the downloaded file on your Android device
3. If prompted, tap **"Allow from this source"** to allow installation
4. Tap **"Install"**
5. Done! ✅

> 💡 The "Allow from this source" prompt only appears **once** on first install.

---

## 🔄 How to Update

When a new version is available, you will see an **"Update Available"** notification inside the app.

1. Tap **"Update Now"** in the app
2. You will be redirected to download the new APK
3. Install it the same way as above ✅
